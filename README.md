# tsg-usa / tsgabrielle MACH Commerce Starter

State-of-the-art foundational scaffold for a 2026-standard headless e-commerce platform using:
- Next.js SSR storefront (`apps/web`)
- Node.js agent orchestrator (`services/agent-orchestrator`)
- Python RAG support microservice (`services/support-rag`)
- Docker + Cloud Run deployment configuration

## Quick start
```bash
npm install
npm run dev
```

See `docs/architecture.md` for the full system design, route map, and agent ecosystem.
